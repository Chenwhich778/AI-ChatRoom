# 截图说明

## 📸 如何添加项目截图

将您的应用截图放在此目录下，建议命名为：

- `login.png` - 登录界面截图
- `main-window.png` - 主界面截图
- `multi-rooms.png` - 多聊天室标签页截图
- `ai-assistant.png` - AI助手功能截图

## 📐 推荐尺寸

- 宽度: 800-1200px
- 格式: PNG 或 JPG
- 文件大小: < 500KB （建议压缩图片）

## 🎯 截图技巧

1. **使用 Windows 截图工具**:
   - 按 `Win + Shift + S` 截取屏幕区域
   - 或使用 `Snipping Tool`

2. **使用 Qt Creator**:
   - 运行应用后截取窗口

3. **美化建议**:
   - 选择有代表性的聊天内容
   - 确保界面完整显示
   - 避免包含敏感信息（API密钥等）

## 🌐 在线图床（可选）

如果不想把大图放在仓库中，可以使用图床：

- [SM.MS](https://sm.ms/) - 免费图床
- [ImgBB](https://imgbb.com/) - 免费图床
- GitHub Issues - 在 issue 中上传图片，复制链接

然后在 README.md 中使用在线图片链接：

```markdown
![登录界面](https://your-image-url.com/login.png)
```

## ✅ 完成后

截图添加完成后，README 中的图片就会自动显示。如果使用图床，记得更新 README.md 中的图片链接。
